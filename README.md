## 🌟 About Me

I’m **3mran** — founder of **[Mayor Studio](https://mayorstudio.xyz)**, a creative tech lab where **code meets curiosity**.

I build **high-performance web applications**, **game automation systems**, and **developer tools** that solve real problems — not just prototypes.

- 🔭 **Currently building**: Real-time collaborative dashboards, Lua-based game mods, and self-hosted automation stacks
- 🌱 **Core skills**: JavaScript/TypeScript, Node.js, Python, Lua, React, Vue
- 💡 **Philosophy**: Clean architecture > flashy features. Performance > pixels. UX > UI.
- 📫 **Let’s talk**: [Discord](https://discord.gg/mayor) | [Control Panel](https://panel.mayorstudio.host)
